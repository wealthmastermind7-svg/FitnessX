1. Generate Workout Plan
This endpoint generates a workout plan based on the user's goals, fitness level, preferences, and schedule.

Endpoint
POST /generateWorkoutPlan
Request Parameters
Parameter	Type	Description
goal	string	The main fitness goal (e.g., "Build muscle", "Lose weight")
fitness_level	string	Current fitness level (e.g., "Beginner", "Intermediate", "Advanced")
preferences	array	Array of preferred workout types (e.g., ["Weight training", "Cardio"])
health_conditions	array	Array of any health conditions to consider
schedule	object	Contains days_per_week (number) and session_duration (number, in minutes)
plan_duration_weeks	number	Duration of the plan in weeks
lang	string	Language code for the response (e.g., "en" for English)
